C-DSP: Moving Average Filter
A lightweight Digital Signal Processing (DSP) implementation in C, featuring real-time filtering and Python-based data visualization.

### Prerequisites
* `gcc` compiler
* `make`
* `python3` with `pandas` and `matplotlib`

Installation & Execution
1. **Clone the repository**:

   git clone https://github.com/chriskikker/C-DSP-Moving-Average.git
   
   cd C-DSP-Moving-Average

3. **Build and generate data**:
   
   make run
   
5. **Visualize results**:
   
   python3 plot_results.py
